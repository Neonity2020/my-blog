## Markdown写作指南

### 01. What is Markdown？

Markdown is a lightweight mark-language.

### 01. 什么是 Markdown ？

Markdown 是一种轻量级的标记语言

### 02. Why Markdown？

Markdown is suitable for network content creation.

### 02. 为什么要使用 Markdown？

Markdown 适合用于互联网内容创作。

### 03. Markdown basic syntax

#### 03-1. Title

```markdown
# Title I

## Title II

### Title III
```

#### 03-2. bullets list

```markdown
- item 01
- item 02
```

#### 03-3. ordered list

```markdown
1. item 01
2. item 02
3. item 03
```

#### 03-4. task list

```markdown
- [ ] task 01
- [ ] task 02
- [x] done 03
```

### 03. Markdown 的基本语法

#### 03-1. 标题

```markdown
# 一级标题

## 二级标题

### 三级标题
```

#### 03-2. 无序列表

```markdown
- 项目 01
- 项目 02
```

#### 03-3. 有序列表

```markdown
1. 项目 01
2. 项目 02
3. 项目 03
```

#### 03-4. 任务列表

```markdown
- [ ] ToDo 01
- [ ] ToDo 02
- [x] Done 03
```
